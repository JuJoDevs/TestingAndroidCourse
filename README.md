# CursoTestingAndroid

Android repository created for educational purposes to practice testing concepts, techniques, and tools.

This project is related to the first edition of the AristiDevs course:
[Testing en Android: Curso definitivo desde 0 (Testing on Android: The definitive course from scratch)](https://www.appcademy.dev/testing-en-android-curso-definitivo-desde-0)

## Purpose

It is intended to support learning around Android testing in a realistic project setup, including different application layers and common development scenarios.

## Notice

This repository is exclusively educational in nature. Its goal is to support the course learning process and make experimentation, practice, and understanding of the covered content easier.
